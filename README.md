# Text Analyzer
Ett enkelt Pythonprogram som analyserar textfiler.  
Programmet läser en textfil, räknar antal ord, antal rader och hittar det vanligaste ordet.

## Installation
1. Skapa och aktivera virtuell miljö:
   - **Windows:** 
     ```
     venv\Scripts\activate
     ```

